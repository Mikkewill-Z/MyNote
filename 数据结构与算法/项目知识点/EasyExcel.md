# 使用
easyExcel是一行一行进行读取,避免了文件读取的错误
### 读数据
	1. 添加对应依赖
	2. 定义实体类封装数据
	3. 定义监听器监听解析到的数据
	4. 编写测试方法
### 添加数据到Excel
监听器是用来进行读取操作的,将Excel中的数据一行一行进行读取
读取之后将数据存储到list集合中
