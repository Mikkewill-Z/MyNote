# 介绍
1.概述:表示特定的瞬间，精确到毫秒
2.常识:
  a.1000毫秒 = 1秒
  b.北京经纬度: 东经116  北纬39.56
  c.气候:温带大陆性季风气候
  d.时区:一个时区差一个小时
  e.0度经线:本初子午线
  f.时间原点:1970年1月1日0时0分0秒
  g.我们所在时区为东八区,比时间原点差8个小时  
# 使用
构造:
  Date()->获取当前系统时间
  Date(long time)->获取指定时间,从时间原点开始算,传递毫秒值
# 常用方法
long getTime()-> 获取当前时间对应的毫秒值
void setTime(long time) -> 设置时间,从时间原点开始算起
